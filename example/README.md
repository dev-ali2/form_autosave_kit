# form_autosave_kit_example

Example application for the `form_autosave_kit` package.

## Getting Started

To run this example project:

```bash
cd example
flutter run
```

This app demonstrates three ways to use `form_autosave_kit`:
* **Checkout Form**: Demonstrates how to use `AutosaveForm`, `AutosaveField` (with TextFields), and `AutosaveDraftBanner` together in a single flow.
* **Registration Form**: Shows how to use the package with non-text fields like `CheckboxListTile` and `DropdownButtonFormField`.
* **Survey Form**: Demonstrates direct use of the programmatic `AutosaveController` interface to manage and inspect drafts manually without the banner UI.
